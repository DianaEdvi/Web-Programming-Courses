HTML Cheat Sheet
Heading: <h1></h1>
Paragraph: <p1></p1>
Horizontal rule: <hr>
Break: <br>
Unordered List: <ul></ul>
Ordered List: <ol></ol>
List element: <li></li>
Anchor element: <a tag attribute=value>Content</a>
Global attribute: <a draggable=true>This is a link</a>
Image Element: <img src="url/size" alt "image name"/>
Webpage: <a href="this/is/a/path.html"><Link name</a>
Webpage with image: <a href="this/is/a/path"<img src="/path/to/image" alt="image name"></a>
Boilerplate intellij: html:5 + Enter
